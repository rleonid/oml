Principle Components Analysis.
type t 
val variances : t -> float array
variances t returns a vector of the amount of variance explained by each PC in t.
val components : t -> float array array
components t
val scalings : t -> (float * float) array
scalings t returns an array of tuples representing the subtracted mean and divided standard deviation applied to the data before broken into PC's.
val pca : ?demean:bool -> ?scale:bool -> ?unbiased:bool -> Lacaml.D.mat -> t
pca data break down data into its principle components.
type pca_reduction_method =
| SpecificNumberOfComponents of int (*
Only use this many components.
*)
| VarianceExplained of float (*
Choose as many components as needed to explain this much variance.
*) 
val reduce : t -> pca_reduction_method -> t
reduce t prm restrict the number of stored principle components via prm.
