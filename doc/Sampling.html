Module Sampling

module Sampling: sig .. end
Create generators for sampling from specified distributions.
type 'a generator = unit -> 'a 
val uniform_i : ?seed:int array -> int -> int generator
uniform_i ?seed n creates a generator that will return an integer with equal probabilities from the interval [0,n).
Raises Invalid_argument if n is zero or negative, or above 2^30 - 1.
val uniform_f : ?seed:int array -> float -> float generator
uniform_f ?seed n creates a generator that will return a float with equal probabilities from the interval [0,n].
Raises Invalid_argument if n is zero or negative.
val normal : ?seed:int array ->
       mean:float -> std:float -> unit -> float generator
normal ?seed mean std () creates a generator that will return variables from the Normal distribution of mean and std (standard deviation).
Raises Invalid_argument if std is less than zero.
val normal_std : ?seed:int array -> unit -> float generator
normal_std seed () is equivalent to normal seed ~mean:0.0 ~std:1.0 ().
val multinomial : ?seed:int array -> float array -> int generator
multinomial ?seed weights creates a generator that will return an integer representating the ith element from the Multinomial distribution given by a weights vector which sums to 1.
Raises Invalid_argument if weights do not sum to 1.0 (this is checked using Util.significantly_different_from) or any individual weight is not in [0,1).
val softmax : ?seed:int array ->
       ?temperature:float -> float array -> int generator
softmax ?seed ?temperature weights creates a generator that will return an integer representating the ith element from the softmax distribution given by a weights vector and temperature parameter which defaults to 1.0.
Raises Invalid_argument if weights is empty or temperature = 0.
module Poly: sig .. end
Provides polymorphic versions that sample over arrays of any type.
