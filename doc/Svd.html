Module Svd

module Svd: sig .. end
Perform the Singular Value Decomposition via Lacaml.
type t

Accessors


val u : t -> Matrices.t
u t
val s : t -> Vectors.t
s t singular values.
val vt : t -> Matrices.t
vt t.
val svd : Lacaml.D.mat -> t
svd m separate m into its singular value decomposition. mat is destroyed in the process.
val solve_linear : ?lambda:float -> t -> Lacaml.D.vec -> Lacaml.D.vec
solve_linear a b solve for x in a x = b using principal components regression, when a is the Singular Value Decomposition.
val covariance_matrix : ?lambda:float -> t -> Lacaml.D.mat
covariance_matrix a b computes the covariance matrix A * A^t.
val looe : t -> Lacaml.D.vec -> float -> Lacaml.D.vec
looe t y returns a function to efficiently computes the Leave-One-Out-Error vector, for computing the optimal regularization parameter.
val h : t -> Lacaml.D.mat
h t computes the Hat matrix of the original matrix passed to Svd.
val h_diag : t -> Lacaml.D.vec
h_diag t computes the diagonal of the Hat matrix of the original matrix passed to Svd.
