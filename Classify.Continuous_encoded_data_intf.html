Module type Classify.Continuous_encoded_data_intf

module type Continuous_encoded_data_intf = sig .. end
Data where features are encoded using a set of quantitative variables.

Each feature is described using an array of quantitative variables of length size. In many cases this is the simplest encoding.

include Classify.Data_intf
val encoding : feature -> float array
encoding f returns an array where each element e at position i describes the ith variable with value e.
Raises Invalid_argument if the returned array does not equal size.
val size : int
The number of quantitative variables needed to describe the features.
