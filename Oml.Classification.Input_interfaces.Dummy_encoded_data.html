Module type Oml.Classification.Input_interfaces.Dummy_encoded_data

module type Dummy_encoded_data = sig .. end
Data where features are encoded using Dummy Variables (aka one-hot encoding).

Each feature is described using an array of binary variables of length size.

include Oml.Classification.Input_interfaces.Data
val encoding : feature -> int array
encoding f returns indices into an array. If i is in encoding f that indicates the presence of dummy variable i in f.
val size : int
The number of dummy variables to track.
